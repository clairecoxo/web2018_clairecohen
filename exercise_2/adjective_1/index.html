Project: An Industry Awakening

A collaboration with the New School Free Press

To Note: I did not do this project on my own but I think that everyone who did partcipate in it's creation was really touched by what we created.

We wanted to include pull quotes to help move the user through the piece.

This element allowed for the user to hover each material and learn more about it's enviromental impact as well as how it is collected.

We included some illustrations—Yasmin made this!—to make the piece a bit more fun. We also thought this would be a good medium in which to add more information and make it more palatable then a block of text.
